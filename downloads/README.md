## 如何添加文件

1. 将对应文件放入上表指定的路径
2. 文件名必须与上表完全一致（包括中文和后缀）
3. 上传到 GitHub 仓库后，Cloudflare Pages 会自动部署
4. 网站上的下载按钮会自动指向对应文件

## 文件大小限制

- 单个文件最大 **25MB**（Cloudflare Pages 硬性限制）
- 超过 25MB 的文件请存放到云盘，在网站上放外链
- 建议整个 downloads/ 目录控制在 200MB 以内

## 文件命名规则

- 使用 `序号-简称.格式` 的命名方式
- 序号按添加顺序递增
- 简称用中文，简明扼要
- 格式后缀：.pptx / .docx / .pdf / .zip
